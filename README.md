# Forum-T (Web Application)
# Front End Techonology: React JS
# Back End Techonology: Node JS
# Database: MySQL
# API's Created
-> SignUp

-> Login

-> (Admin)

  _Get all User Info
  
  _Get all Post

  _Get all feedback
  
  _Make User as Admin
  
  _Remove User from Admin
  
  _Make User Inactive (Soft Delete)
  
  _Make User Active
  
  _Delete Post
  
  _Delete Feedback
  
-> Fetch all Post

-> Add new Post

-> Add new Feedback
